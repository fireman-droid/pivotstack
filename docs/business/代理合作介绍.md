# 代理合作介绍

## 我们是谁

我们提供 Claude AI 模型 API 中转服务，支持 Claude Code 等主流开发工具直接接入。

## 代理模式

你将获得一个**主 Key**，拥有独立的代理面板，可以：

- ✅ 自主创建子 Key，分发给你的客户
- ✅ 给每个子 Key 分配额度，花完了再充
- ✅ 查看每个客户的用量、余额、请求记录
- ✅ 随时启用/禁用/删除子 Key

**你的客户用你的子 Key 调用 API，和直接用我们平台完全一样。**

## 操作流程

1. 联系我们开通代理主 Key
2. 充值到你的主 Key
3. 在代理面板创建子 Key（可以加备注，比如"客户A"）
4. 从主 Key 余额划转额度到子 Key
5. 把子 Key 发给客户，客户直接使用

## 代理面板功能

| 功能 | 说明 |
|------|------|
| 子 Key 管理 | 创建、删除、启用/禁用 |
| 额度划转 | 从主 Key 余额转到子 Key |
| 用量查看 | 每个子 Key 的请求数、token 消耗 |
| 总览面板 | 主 Key 余额、子 Key 数量、总消耗 |

## 隐私保障

- 🔒 **你的客户看不到我们** — 客户只知道从你这里拿 Key，看不到后端信息
- 🔒 **子 Key 由你管理** — 创建、分配、删除都是你自己操作
- 🔧 **技术支持** — 管理员后台可以看到子 Key 数据，用于排查技术问题，但不会主动联系你的客户
