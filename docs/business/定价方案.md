# Kiro 中转站定价方案

## 成本结构

- 账号成本：¥60 / 1500 credits
- 1 credit = ¥0.04

## Token 换算公式（实测）

```
credit = 0.0036268 × input_tokens + 0.0001092735 × output_tokens + 0.00948186
```

| 单位 | Credits | 成本（元） |
|------|---------|-----------|
| 1K input tokens | 0.003627 cr | ¥0.000145 |
| 1K output tokens | 0.109273 cr | ¥0.004371 |
| 每次请求固定开销 | 0.009482 cr | ¥0.000379 |
